# Stationery Inventory Management System

## About
This is a Python project created for the Programming 2 module (SF43002FP).
The application was built for a fictional client called StationeryTrack Pte Ltd to help them manage their stationery stocks.

## Files
- `project 2.py` — the main script that runs the application
- `stationery.py` — a separate class file that calculates total price of each item

## Features
- **Option 1** — Add new stationery items
- **Option 2** — Edit existing stationery items
- **Option 3** — Update quantity of items sold
- **Option 4** — Display all items in table format
- **Option 5** — Save all items to a CSV file

## Libraries Used
- `csv` — to save stationery items into a CSV file
- `os` — to clear the screen after each action

## How To Run
1. Make sure Python 3.0 or higher is installed
2. Make sure both `project 2.py` and `stationery.py` are in the same folder
3. Run `project 2.py` in your Python editor
4. Follow the menu options on screen
